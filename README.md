# Local Service Booking Platform

A full-stack web application that connects users with local service providers such as plumbers, electricians, and tutors.

## Tech Stack
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: JWT
- Tools: Git, GitHub

## Features
- User and service provider authentication
- Role-based access (user / provider)
- Service listing and booking system
- Secure password hashing
- RESTful APIs
